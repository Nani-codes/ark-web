## 🏗️ About

ARK Assured is a premium marketplace platform that eliminates counterfeits and middlemen from the building materials supply chain. We connect verified brands directly with construction professionals and homeowners, ensuring 100% genuine products at institutional pricing.

## ✨ Features

- **Zero Counterfeits**: 100% verified brand-direct sourcing
- **Direct Sourcing**: Eliminate middlemen and save up to 30% on materials
- **Dual Interface**: Separate platforms for professionals (B2B) and homeowners (B2C)
- **Verified Network**: Connect with ARK-verified designers and contractors
- **Physical Experience Centers**: Touch and feel materials before purchase
- **Express Logistics**: Direct brand-to-door delivery with full tracking

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Yash-d21/ARK_Land.git
   cd ark-assured---the-gold-standard-for-building-materials
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
